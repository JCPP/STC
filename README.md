STC
===

## Che cos'è STC? ##
**Statistiche Territoriali Climatiche** è un progetto per Arduino che ha come scopo principale quello di monitorare il clima di un territorio, in modo da creare statistiche, grafici e foto nel corso dell'anno che potranno essere utili per fini di studio meteorologico.
Oltre a questo il progetto contiene anche dei sistemi di alert che permettono di controllare una determinata situazione meteorologica e di avvertire coloro che ne hanno richiesto il servizio nel caso in cui ci sia una situazione di pericolo (esempio: temperatura troppo alta, vento forte o alto rischio di fulmini).


## Come collaborare per il progetto ##
Innanzitutto è necessario seguire la [guida ufficiale del plugin di Eclipse](http://www.baeyens.it/eclipse/Install.html) che permette di lavorare con Arduino sfruttando le potenzialità di un IDE avanzato.
Si deve fare in modo che Eclipse riesca a riconoscere i file *.ino* tipici di Arduino, e per far questo bisogna utilizzare [questa guida](http://www.baeyens.it/eclipse/IDECompatibility.html).
Per altre informazioni è necessario dare un'occhiata alla home page del [plugin](http://www.baeyens.it/eclipse/) (in basso si possono trovare altri link interessanti).
